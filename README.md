# Ucool - Online POS for Small Businesses

**Ucool**, Developed as an intensive MVP in just one month for my FDP, is a cloud-based Point of Sale (POS) solution designed to help small business owners digitize their sales process, maintain accurate inventory control, and professionalize their billing system through an intuitive and modern interface.

---

## 🚀 Core Features

-   **Inventory Management:** Real-time tracking of products, categories, and stock levels.
-   **Sales Management:** Fast and responsive checkout interface designed for daily operations.
-   **Invoicing:** Automated generation of digital invoices and receipts for customers.
-   **Cloud-based:** Accessible from anywhere, ensuring your business data is always synchronized.

---

## 🛠️ Tech Stack

This project is built using the **MERN** stack for high performance and scalability:

-   **Frontend:** [React.js](https://reactjs.org/) with Hooks and Context API.
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/) for a modern, utility-first responsive design.
-   **Backend:** [Node.js](https://nodejs.org/) & [Express](https://expressjs.com/).
-   **Database:** [MongoDB](https://www.mongodb.com/) with Mongoose for flexible data modeling.
-   **Authentication:** Secure user access via JSON Web Tokens (JWT).
